# Como Baixar e Imprimir

## Pelo computador

1. Entre no repositório pelo GitHub.
2. Abra a pasta `imagens`.
3. Clique na arte que deseja baixar.
4. Clique em `Download raw file`.
5. Salve o arquivo.
6. Imprima em casa ou envie o arquivo para uma gráfica.

## Pelo celular

1. Abra o link do repositório.
2. Toque na pasta `imagens`.
3. Toque no arquivo escolhido.
4. Use a opção de download ou compartilhamento do navegador.
5. Salve a imagem no celular.

## Dica

Se for imprimir em gráfica, envie o arquivo original baixado do GitHub, sem tirar print da tela. Assim a qualidade fica melhor.
