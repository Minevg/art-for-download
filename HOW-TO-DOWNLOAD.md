# How to Download and Print

## On a Computer

1. Open the repository on GitHub.
2. Open the `images` folder.
3. Click the artwork you want to download.
4. Click `Download raw file`.
5. Save the file.
6. Print it at home or send it to a print shop.

## On a Phone

1. Open the repository link.
2. Tap the `images` folder.
3. Tap the selected file.
4. Use your browser download or share option.
5. Save the file to your phone.

## Tip

If you are printing at a print shop, send the original file downloaded from GitHub instead of a screenshot. This keeps the quality much better.
